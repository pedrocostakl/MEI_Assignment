 400 |     def run_sync(self, func, timeout=None):
 401 |         """Starts the `IOLoop`, runs the given function, and stops the loop.
 402 | 
 403 |         If the function returns a `.Future`, the `IOLoop` will run
 404 |         until the future is resolved.  If it raises an exception, the
 405 |         `IOLoop` will stop and the exception will be re-raised to the
 406 |         caller.

 408 |         The keyword-only argument ``timeout`` may be used to set
 409 |         a maximum duration for the function.  If the timeout expires,
 410 |         a `TimeoutError` is raised.

 413 |         to allow asynchronous calls in a ``main()`` function::
 414 | 
 415 |             @gen.coroutine

 420 |                 IOLoop.current().run_sync(main)

 424 |         def run():
 425 |             try:
